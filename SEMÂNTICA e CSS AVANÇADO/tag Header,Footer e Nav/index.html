Tag Header, Footer e Nav

Essas tags não mudam nada visulamente

A tag header é a tag que cuidará do cabeçalho de nossa aplicação.

Para realização de um menu tem que ser dentro da tag nav, facilita para pessoas com deficiência auditiva e visual

A tag Nav é a tag que cuidará dos menus de navegação que a nossa aplicação terá.

A tag Footer é a tag que cuidará do rodapé de nossa aplicação
