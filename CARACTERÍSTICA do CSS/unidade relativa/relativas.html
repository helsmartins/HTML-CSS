Unidade de medida (rem)
Todos os outros elementos devem ser descendentes desse elemento

Unidade Porcentagem (%)
A unidade de porcentagem no CSS é relativa ao bloco onde o elemento está contido. É ótimo para imagens: aqui mostramos uma imagem que sempre terá 50% da largura de seu container.

Unidade relativa (viewport)
As unidades relativas vw (viewport width) e vh (viewport height) utilizam a largura e altura da viewport respectivamente. Com elas é possível atribuir a um elemento o tamanho (seja altura, seja largura) da viewport. Essa unidade de medida é muito parecida com a unidade relativa de porcentagem % .
